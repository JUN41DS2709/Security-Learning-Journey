## About

This repository documents my cybersecurity learning journey as I work toward becoming a penetration tester and offensive security professional.

The goal of this repository is to maintain consistency, track progress, document key concepts, and build a public record of my hands-on learning experience.

Rather than focusing only on certifications, I want to develop practical skills through continuous learning, labs, projects, and real-world problem solving.
